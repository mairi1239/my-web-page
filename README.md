The image above is the result of running the following HTML code 

    <!DOCTYPE html>
    <html>
    <head>
    <title>CSS Color Challenge</title>

    <!-- Linking the external CSS file -->
    <link rel="stylesheet" href="style.css">
    </head>
    <body>

    <h1>CSS Color Challenge</h1>

    <p class="purple-text">Explore the spectrum of colors!</p>

    <p class="yellow-box">Hello world, how are you?</p>

    <p class="green-box">Unstyled Paragraph</p>

    </body>
    </html>


the colors and shapes in the image are made with the following CSS code

    /* Blue heading */
    h1 {
    background-color: #2196F3; /* bright blue */
    color: blue;
    padding: 15px;
    }

    /* Purple text */
    .purple-text {
    color: red;
    }

    /* Yellow bar */
    .yellow-box {
    background-color: yellow;
    color: orange;
    padding: 10px;
    }

    /* Light green bar */
